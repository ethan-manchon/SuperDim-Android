# SuperDim

Une application Android simple pour réduire la luminosité de l'écran en dessous du minimum natif en utilisant un overlay sombre.

## Fonctionnalités
- Réduction de la luminosité via un overlay ajustable.
- Activation/désactivation rapide.
- Ajustement de l'opacité de l'overlay.

## Prérequis
- Android 8.0 (API 26) ou supérieur.
- Autorisation "Afficher par-dessus les autres apps" (SYSTEM_ALERT_WINDOW).

## Installation
1. Téléchargez l'APK depuis la section [Releases](https://github.com/ethan-manchon/SuperDim-Android/releases).
2. Installez l'APK sur votre appareil.
3. Activez l'autorisation "Afficher par-dessus les autres apps" dans les paramètres Android.

## Utilisation
- Ouvrez l'application.
- Ajustez le curseur pour modifier l'opacité de l'overlay.
- Appuyez sur le bouton pour activer/désactiver l'overlay.
